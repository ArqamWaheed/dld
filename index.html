1. Number Systems & Conversions

Q1.

Convert (101101.11)2 to decimal.

Q2.

Convert (255)10 to binary, octal, and hexadecimal.

Q3.

Group binary digits to convert (11010110)2 to octal and hexadecimal directly.

Q4.

Q5.

2. Binary Codes & Logic

Q6.

Express 394 in BCD. Add BCD 0111 + 0110 and correct if needed.

Q7.

Convert decimal 6 to Gray code and explain why only one bit changes per step.

Q8.

A 7-bit ASCII character has even parity. If 3 bits are 1, what is the parity bit?

Q9.

3. Complements & Subtraction

Q10.

Find the 1's complement and 2's complement of (10110100)2.

Q11.

Find the 9's complement and 10's complement of 3728.

Q12.

Subtract using 2's complement: 1010 − 0110. Identify the end carry.

Q13.

Subtract using 1's complement: 1010 − 1100. Apply end-around carry if present.

Q14.

Subtract 10's complement: 52 − 79. What is the sign and value of the result?

Answer Key

  1. Q1. 32 + 8 + 4 + 1 + 0.5 + 0.25 = 45.75
  2. Q2. Binary: 11111111  |  Octal: 377  |  Hex: FF
  3. Q3. Group 3s → 011 010 110 = (326)8; Group 4s → 1101 0110 = (D6)16
  4. Q4. 0.6875 × 2 → 0.1011 → (0.1011)2
  5. Q5. 1·r + 3 = 7 → r = 4
  6. Q6. 394 in BCD: 0011 1001 0100.  0111 + 0110 = 1101 (invalid) → add 0110 → 0001 0011 (carry + 3)
  7. Q7. Decimal 6 = 0110 binary → XOR with shifted = 0101 Gray. Adjacent codes differ by exactly 1 bit (unit-distance property).
  8. Q8. Three 1s are odd; need one more 1 to make even → parity bit = 1
  9. Q9. AND = 0  |  OR = 1  |  NOT(A) = 0
  10. Q10. 1's complement: 01001011  |  2's complement: 01001100
  11. Q11. 9's complement: 9999 − 3728 = 6271  |  10's complement: 6272
  12. Q12. 2's comp of 0110 = 1010.  1010 + 1010 = 10100. End carry discarded → result = 0100 (+4 ✓)
  13. Q13. 1's comp of 1100 = 0011.  1010 + 0011 = 1101. No end carry → take 1's comp → 0010, result = −0010 (−2 ✓)
  14. Q14. 10's comp of 79 = 21.  52 + 21 = 73, no end carry → result is in complement form → 10's comp of 73 = 27, result = −27
